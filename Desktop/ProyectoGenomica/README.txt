Para ejecutar el proyecto es necesario tener instalado : 

- numpy : 
    pip install numpy
- matplotlib : 
    python -m pip install -U pip
    python -m pip install -U matplotlib 
- scipy :
    pip install scipy


Para la ejecución : 

    python3 Proyecto.py a.txt x

con x = 

    1 : Para ver la gráfica de proporción de contenido A C T G 
    2 : Arbol Filogenético UPGMA
    3 : Arbol Filogenético WPGMAD

Ejemplo de ejecución : 

    python3 Proyecto.py a.txt 2
